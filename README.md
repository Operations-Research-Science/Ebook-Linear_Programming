## 出品团队

『运筹OR帷幄』公众号作为华人最大的运筹优化|DS|AI社区之一，有着强大的技术背景和丰富的企业资源。公众号以运筹学/优化为基石，积极拓展数据科学/人工智能等相关领域，致力于打造精准&高度垂直的算法和数据社区，聚集领域内优秀学者、企业和优质人群。

2019年1月，我们在公众号发布了线性规划入门电子书的征稿文，反馈影响热烈，最终由以下同学组成作者团队，共同完成了本书的创作。
